Thank you for downloading!

To install:
1. Drag the "CWW-RecolorPack-UK-BronzeGreen" folder from inside "Place in MODS version folder" into <Your WoT Directory>\mods\<version>. (EXAMPLE: C:\Games\World_of_Tanks\mods\1.0.2.4)
2. Drag the "vehicles" folder from inside "Place in RES_MODS version folder" into <Your WoT Directory>\res_mods\<version>. (EXAMPLE: C:\Games\World_of_Tanks\res_mods\1.0.2.4)

Optional:
- If you want a remodel for the Crusader which adds the sideskirt plates it had as an option historically, drag the .wotmod file from the "ADDON - Crusader with Sideskirts Remodel (by Milkym4n)" folder to <Your WoT Directory>\mods\<version> (EXAMPLE: C:\Games\World_of_Tanks\mods\1.0.2.4).

- If you want to also remove the skin from the Cromwell Berlin, drag the .wotmod file from inside the "ADDON - Plain Cromwell B Skin (by Avalon304)" folder to <your WoT folder>\mods\<version> (example: C:\Games\World_of_Tanks\mods\1.0.2.4). Thanks to Avalon304 for the original mod.

- Additionally, if you want to apply countershading to the Firefly's barrel, drag the "vehicles" folder from "ADDON - Firefly Barrel Countershading (by Desyatnik_Pansy)" to <Your WoT Directory>\res_mods\<version> (EXAMPLE: C:\Games\World_of_Tanks\res_mods\1.0.2.4). Thanks to Desyatnik_Pansy for providing the original files.

Enjoy!
